# Rinoimob App (Portal Interno)

Vue 3 + Vite - Internal admin dashboard for property and lead management.

## Tech Stack
- Vue 3
- TypeScript
- Vite
- TailwindCSS
- Pinia (State Management)

## Getting Started

```bash
npm install
npm run dev
```

## Project Structure
- `src/components/` - Vue components
- `src/pages/` - Page components
- `src/stores/` - Pinia stores
- `src/api/` - API client

## Features
- Property Management
- Lead Pipeline
- Admin Dashboard
- User Management
